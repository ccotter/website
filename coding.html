Deterministic Linux

The largest and most challenging project I have worked on was making Linux deterministic. Inspired by the design of another deterministic operation system described by Aviram et al.'s in Efficient System-Enforced Deterministic Parallelism paper, I modified the Linux kernel to support deterministic execution of user programs. I extended the kernel syscall API adding three new syscalls for deterministic processes. I wrote a replacement for the standard C library that wraps around the new syscall interface to provide familiar functionality to application programmers.

Read my undergraduate thesis, Enforced Deterministic Program Execution in the Linux Kernel and check out the source code for the modified kernel (see the v3.0-det branch) and user library on Github.

OpenCoweb

OpenCoweb is an application framework that connects remote clients to a cooperative session on the Web. Application programmers use JavaScript to write to send cooperative events to remote clients and the framework
